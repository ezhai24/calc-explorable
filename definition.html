Slopes

Let's start simple. Given two points, can you calculate the slope?

The slope of a line, denoted by \(m\), is given by the formula $$m = \frac{\textrm{change in y}}{\textrm{change in x}} = \frac{y_2 - y_1}{x_2 - x_1}$$ So in this case the slope is 2.
Given this defintion, it should be apparent that changing the position of the coordinates changes the slope. Try moving the points around and watch how the slope changes!
Ok. Now let's get a little crazy. We'll reset the points and run a curve through them.
Can you calculate the slope between the two points on the curve?

The slope of the line, is still 2!
You can still move the points around too. Try it! You may notice there's a bit of a difference. The points can only move along the curve now. That's because in order for us to say we're calculating the slope between two points on the curve, the two points must stay on the curve. Pretty intuitive right?
Now try moving the points really reeaaallly close together until they're on top of each other.
... looks like we broke the code... but I'll get back to that. First, we need to take a short detour.
$$m = \frac{9.0-1.0}{5.0-1.0} = 2.0$$

Limits

There's a lot of complexity and nuance to limits but we're not going to worry about that here. All we need to understand is the basic idea which is, lucky for you, incredibly simple.
To help explain we'll start with just one point this time. Try moving the circle closer to the X.
The closer the circle gets to the X, the bluer it gets! We can express this sentiment more formally by saying "The limit of the color, as the circle approaches the X, is blue" and write it mathematically like so: $$\lim_{circle \to X} color(circle) = blue$$ Note: we write color(circle) because color is a function of the position of the circle. In other words, the color of the circle depends on it's position.
And that's a limit! Limits aren't actually mathematical by nature. They just describe a variable and the value it approaches as some other variable it depends on approaches another value.
